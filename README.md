# Squadron 3D Game
Airplanes War game, based on Unity 3D game engine.

# Game Architecture
This project built using MVP design patterns and other patterns like Messaging Pattern, Repository Pattern and others.

# Game Featuers
* Multiple Airplanes.
* Airplanes Store.
* Rank System.
* User Profile.
* Dynamic Game Difficulty.
* Bot System.
* Save/Load System
* Score/Coins System.
